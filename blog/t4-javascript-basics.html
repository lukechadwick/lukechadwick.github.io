As if talking to a non-tech friend, discuss:

27 June 2018

How does JavaScript compare to HTML and CSS?
HTML is just the basic content and structure of a website. CSS tells the browser about the design and how to display these elements, like colors, shapes and sizes. Javascript let's us make the page interactive and perform tasks and functions that modify the data either locally or on a server.
Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.
Waking up could be controlled by an if statement, such as 'is alarm or other noise sounding', 'if sleep (is enough) then wake'. Bushing teeth could use a for loop with a certain amount of cycles before running an if statement to check if they are clean enough and repeating the proccess if not.
Explain the difference between accessing data from arrays and objects.
Arrays are sets of ordered data that are accessed by their index (integer) starting at 0. Objects are more granular in that they store data using a key-value. The data can also be different types where as arrays are better for storing the same type of data.
Explain what functions are and why they are useful.
Functions let us create a set of instructions that can be reused with different sets of data, processing it and returning a result, this can save code and time.
