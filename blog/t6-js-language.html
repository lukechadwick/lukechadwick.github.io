Explain to a non-tech friend:

5 July 2018

In the JavaScript language, describe all the use cases for each of:
JavaScript uses parenthesis in many cases, some examples are to group a sum like x = 20 - (5/2). Another example is for functions where the arguments go inside, such as updateMoney(amount, account), they can also be empty if the function does not require any arguments.
brackets []
Brackets have a few uses such as accessing an objects properties [prop], or for indexes [i]
braces {}
Braces are used to create an empty object object = {}, or to contain code segments of functions functionDoStuff(arguments) { code }.
"single quotes '' and double quotes ""
These can usually be used interchangeable to create a string, though certain strings can have issues depending on the content and may need an escape character to be used so it's up to the discretion of the programmer when it's best to use them.
